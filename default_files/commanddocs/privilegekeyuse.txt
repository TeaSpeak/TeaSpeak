Usage: privilegekeyuse token={tokenKey}

Permissions:
  b_virtualserver_token_use

Description:
  Use a token key gain access to a server or channel group. Please note that
  the server will automatically delete the token after it has been used.

Example:
  privilegekeyuse token=eKnFZQ9EK7G7MhtuQB6+N2B1PNZZ6OZL3ycDp2OW
  error id=0 msg=ok
